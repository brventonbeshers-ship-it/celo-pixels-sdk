# celo-pixels-sdk

TypeScript SDK for interacting with the CeloPixels contracts on Celo.

## Installation

```bash
npm install celo-pixels-sdk
```

## Usage

```ts
import { CeloPixelsClient } from "celo-pixels-sdk";

const client = new CeloPixelsClient({
  contractAddress: "0x..."
});
```

## Development

```bash
npm install
npm run build
```
